## 🎯 About Me

I'm passionate about **LLMs, Agent Systems, and Reinforcement Learning**. I believe the best way to understand complex concepts is to build and teach them clearly.

**Current Focus:**
- 🚀 **LLM Architectures** — MoE, Attention variants, Long Context
- 🧠 **Reinforcement Learning** — GRPO, Test-Time Compute Scaling
- 🤖 **Agent Systems** — Multi-Agent, Tool Use, Planning
- ⚡ **Inference Optimization** — Speculative Decoding, Quantization, KV Cache
