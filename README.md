## 🧑‍💻 About Me

I'm passionate about **pushing the boundaries of AI** — from large language models to reinforcement learning, agent systems to inference optimization. I believe the best way to understand complex concepts is to teach them clearly.

My work focuses on:
- 🚀 **LLM Architectures** — MoE, Attention variants, Long Context
- 🧠 **Reinforcement Learning** — GRPO, PPO, Test-Time Compute Scaling
- 🤖 **Agent Systems** — Multi-Agent Reflection, Tool Use, Planning
- ⚡ **Inference Optimization** — Speculative Decoding, Quantization, KV Cache
